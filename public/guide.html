Getting started

Sign in
  • Open the app URL and sign in with your email + password.
  • If your email is listed as a super admin, you’ll see admin features immediately.
Keyboard basics
  • Tab to move between fields, Enter to submit.
  • Use the sidebar to navigate; on mobile, tap the menu icon.

Roles

  • Admin – full access, manage roles.
  • Manager – create/edit content (inventory, products, tasks, COGS).
  • User – read-only by default (unless you elevate in Settings > Users).

Home

Quick tiles to jump into Inventory, Products, COGS, and Tasks. A good daily starting point.

Inventory

Add & edit items

  • Click Add Item to create a new stock row.
  • Each row shows an image, code, type, price, stock, and a low-stock threshold.

Adjust stock

  • Use +/– to change stock quickly.
  • Rows turn amber/red as they near or cross thresholds.

Tip: On phones, tap the image to see a larger preview.

Products

  • Manage sellable items with barcode, price, type, and instructions.
  • Click a product image/card to view details in a modal.

COGS

  • Add daily rows for gross income and costs (produce, items, freight, delivery, other).
  • Totals and gross profit are calculated automatically at the bottom.

Tasks

  • Three lanes: To do, In progress, Done.
  • Drag & drop between lanes—even when a lane is empty.
  • Movement rules: To do ↔ In progress ↔ Done (with sensible restrictions).

If a move is not allowed, you’ll see a notice.

Search

Use the global search box in the sidebar to find anything across posts, inventory, products, COGS, and users.

Settings

  • Cloud Sync: turn ON to sync to Firebase RTDB; turn OFF to stay fully local.
  • Theme: switch mode and font size instantly.
  • Users: (admin/manager only) set roles for emails that sign in.

Offline & Sync

  • Works offline using local storage.
  • When Sync is ON, data lives under /tenants/{uid}/kv/* in your RTDB and updates live across devices.

Troubleshooting

I can’t sign in
  • Confirm the email exists in Firebase Authentication users list.
  • Reset password if needed (Firebase Auth).
My data doesn’t sync
  • Check Settings → Cloud Sync is ON.
  • Ensure RTDB rules allow auth.uid to its own tenant path.
  • Open the browser console for any errors.
